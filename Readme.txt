=========================================
     PROJECT KLINIK PRATAMA
=========================================

Deskripsi:
-----------
Project Klinik Pratama adalah aplikasi web manajemen klinik yang dibuat menggunakan PHP Native dengan bantuan framework Bootstrap. Aplikasi ini mendukung beberapa role pengguna seperti Pasien (User), Dokter, Admin, dan Apoteker, dengan fitur-fitur yang mendukung digitalisasi layanan di klinik.

Fitur Berdasarkan Role:
------------------------

1. USER / PASIEN:
- Login & Registrasi Akun
- Pengambilan Antrian secara Online
- Upload Bukti Pembayaran
- Memberikan Review (Rating Bintang & Saran)

2. DOKTER:
- Menerima dan Memverifikasi Antrian Pasien
- Membuat Resep Obat untuk Pasien
- Menyusun dan Mengatur Jadwal Dinas Praktik

3. ADMIN:
- Menerima dan Memverifikasi Bukti Pembayaran
- Mencetak Laporan Kegiatan dan Pembayaran
- Broadcast Informasi ke Pasien via WhatsApp Gateway

4. APOTEKER:
- Verifikasi Resep dari Dokter
- Melihat dan Mengelola Laporan Obat

Teknologi:
----------
- PHP Native
- Bootstrap 5
- MySQL Database
- (Optional) FPDF untuk cetak laporan
- WhatsApp Gateway untuk broadcast pesan


Catatan:
---------
Project ini masih dalam tahap pengembangan awal dan dapat dikembangkan lebih lanjut dengan fitur tambahan seperti:
- Integrasi Payment Gateway
- Notifikasi Real-time
- Dashboard Statistik
- Migrasi ke Framework Laravel
