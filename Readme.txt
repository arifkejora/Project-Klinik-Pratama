=========================================
🏥  PROJECT KLINIK PRATAMA
=========================================

📄 Deskripsi:
-------------
Project Klinik Pratama adalah aplikasi web manajemen klinik yang dibuat menggunakan **PHP Native** dengan bantuan framework **Bootstrap**. Aplikasi ini mendukung beberapa role pengguna seperti Pasien (User), Dokter, Admin, dan Apoteker, dengan fitur-fitur yang mendukung digitalisasi layanan di klinik.

👥 Fitur Berdasarkan Role:
--------------------------

1️⃣  USER / PASIEN:
   🔐 Login & Registrasi Akun  
   🧾 Pengambilan Antrian secara Online  
   📤 Upload Bukti Pembayaran  
   ⭐ Memberikan Review (Rating Bintang & Saran)

2️⃣  DOKTER:
   ✅ Verifikasi Antrian Pasien  
   💊 Membuat Resep Obat  
   📅 Menyusun Jadwal Dinas Praktik

3️⃣  ADMIN:
   💰 Verifikasi Pembayaran  
   🖨️ Cetak Laporan Kegiatan dan Pembayaran  
   📢 Broadcast Informasi via WhatsApp Gateway

4️⃣  APOTEKER:
   ✔️ Verifikasi Resep  
   📊 Laporan Ketersediaan dan Distribusi Obat

🛠️ Teknologi yang Digunakan:
----------------------------
- ⚙️ PHP Native  
- 🎨 Bootstrap 5  
- 🗃️ MySQL Database  
- 🧾 FPDF (opsional) untuk cetak laporan  
- 📱 WhatsApp Gateway untuk broadcast pesan


📝 Catatan:
-----------
Project ini masih dalam tahap pengembangan awal dan dapat dikembangkan lebih lanjut dengan fitur seperti:
✨ Integrasi Payment Gateway  
🔔 Notifikasi Real-time  
📊 Dashboard Statistik  
🚀 Migrasi ke Framework Laravel
